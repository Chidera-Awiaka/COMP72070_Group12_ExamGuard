\doxysubsection{Detailed Description}
\begin{DoxyVerb}Wrapper around different "image sources".  You can make images
from a PIL Image object, a filename (in which case it uses PIL),
an image we previously cached (optimisation, hardly used these
days) or a JPEG (which PDF supports natively).\end{DoxyVerb}
