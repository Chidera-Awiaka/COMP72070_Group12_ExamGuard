\doxysubsection{Detailed Description}
\begin{DoxyVerb}This is a generic flag class that all the flags in this file use as a basis.

    This class basically provides edges and a tidy-up routine to hide any bits of
    line that overlap the 'outside' of the flag

    possible attributes:
    'x', 'y', 'size', 'fillColor'
\end{DoxyVerb}
