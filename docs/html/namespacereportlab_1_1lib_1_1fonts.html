A place to put useful font stuff. 

 Font Mappings

The brute force approach to finding the correct postscript font name; much safer than the rule-based ones we tried. preprocessor to reduce font face names to the shortest list possible. Add any aliases you wish; it keeps looking up until it finds no more translations to do. Any input will be lowercased before checking.
